# Arbitrage Detector (MLB)

Monitors MLB odds using TheOddsAPI and notifies you via Discord of arbitrage opportunities.

## Setup

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file using `.env.example`
4. Run: `python main.py`

## Deployment

Deployed to Render as a cron job every 5 minutes using `render.yaml`.
